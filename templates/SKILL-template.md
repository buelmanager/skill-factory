# Skill Name

## Overview

이 스킬이 무엇인지, 핵심 원칙을 1~2문장으로.

## When to Use

- 증상/상황 불릿 목록 (에러 메시지, 증상 키워드 포함)
- 사용하지 **않아야** 할 때도 명시

## Core Pattern

전/후 비교 코드 또는 핵심 패턴.

```
# ❌ Before
...

# ✅ After
...
```

| ... | ... |

## Implementation

간단한 패턴은 인라인 코드로. 무거운 레퍼런스(100줄 이상)나 재사용 도구만 별도 파일로 분리하고 링크.
